## Severity Levels

- **Sniffles**: Sneeze every 5-10 minutes (mild cold)
- **Flu**: Sneeze every 3-7 minutes (moderate cold)
- **Plague**: Sneeze every 2-5 minutes (severe cold)

- `vscold.enabled` - Enable or disable the extension
